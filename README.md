# todo-app

To install packages run `npm install`  
if that doesn't work remove **node_modules** folder and re run the `npm install`

## Dealing with database

First you need to install mongodb on your server  
_make sure that mongodb is active_

## To run the code

To run `npm start`

_This stores your item in the database running locally._  
**open `localhost:3000/` after that type any word or characters `localhost:3000/myitems`. Add your items there**
